chmod 755 sur le répertoire cache et 666 sur les fichiers de ce répertoire
Activez dans le menu administration le module de Chat 
Ensuite vous pouvez activer le bloc correspondant.

Vous disposez dans le bloc de quatres options :
-Hauteur de l'IFRAME en pixel
-Nombres de lignes afffichées
-Nombre de caractères en saisie : taille du champ de saise
-Visualisation seulement : Le bloc ne fait qu'afficher le contenu du chat, ce dernier est accessible via le bouton Popup
